BubbleSort

Bubblesort works in passes through the structure that holds elements. Durin a pass the algorithm starts at one end works its way through each position of the structure. At each position, it tests whether the current element and its neighbour are correctly ordered. If that is not the case, the elements are swapped.

Note that after the first pass, the largest element will be at the very right, where it belongs. This is true, because at each position, the larger of both elements is swapped. At some point during the first pass, the largest element must be swapped with some neighbour. And the again, until it reaches the end. This procedure is the repeated. I.e., after the second pass, the second largest element is correctly placed as second to last element on the right.

This notion of letting the largest elements incrementally rise to their correct place to the right is what gives the algorithm its name. The largest elements "bubble up" through the array.
